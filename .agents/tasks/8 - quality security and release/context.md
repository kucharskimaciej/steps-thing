# Quality, Security, And Release Context

These tasks verify the rewrite is shippable. They should be completed after feature work and kept updated as features evolve.

Required evidence:

- Unit tests for domain algorithms.
- Component tests for critical UI flows.
- Server tests for owner authorization and mutation rules.
- Manual smoke test checklist for auth, upload, create/edit, feed, public sharing, sessions, and migration.
- Production readiness for environment variables, storage access, and observability.
